# Tabs

?> Supported by [bufferline.nvim](https://github.com/akinsho/bufferline.nvim) and [snacks.nvim](https://github.com/folke/snacks.nvim).

Tabs is the tabline at the top of the Neovim editor, shows opened buffers/files.
